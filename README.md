## About The Project
**Job Wizard** is a responsive frontend application built with React and Tailwind CSS that consumes a RESTful API to display and manage job and company data. It connects seamlessly to a Spring Boot backend, using Axios for data fetching and React Router for client-side navigation. The UI is designed to be intuitive, modular, and easily extendable for future features such as filtering, authentication, and admin functionality.
